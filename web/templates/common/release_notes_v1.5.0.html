What's New

Release x.x.x

  • Added the ability to search repos. Examples:

    • repo:test To search all repos with `test` in their name.
    • repo:.* max_matches:6000 To view all indexed repos. Note regex MUST be enabled for the `.*` operator to work.
  • Clarified the use of the overloaded path and repo special query terms in the helparea below. repo:searchTerm and path:searchTerm will look only through repo and file names respectively. Examples included below.
  • Recent Searches and Special Query Terms are shown side-by-side now.

  • Removed the Repo Dropdown Selector. It was causing some usability issues, and removing it enabled us to remove a lot of frontend complexity. The same functionality can be achieved just by using the repo: special term. Here are some examples:

    • repo:eventtracker- help max_matches:200. Will search for the term `help` in all repos starting with `eventtracker-`, with a max of 200 matches.
    • repo:dti-analytics|eventtracker-server help. Will search for the term `help` in repos containing `dti-analytics` OR `eventtracker-server` in their names.
  • General UI Tweaks.
    • Fonts and colors updated. More contrast in helparea items
    • Fixed overflow issue for long code lines without whitespace in search results.
